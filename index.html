Projeto 2

Datalumini

Datalumini é um sistema de monitoramento inteligente que captura a incidência da luz dentro de cultivos de Arabidopsis thaliana.

Desenvolvido em conjunto com alunos da faculdade e a pedido da própria planta, o projeto integra sensores, visualização de dados e banco de dados.

RECURSOS PRINCIPAIS:

  • Captura e análise de intensidade luminosa
  • Interface visual com gráficos em tempo real
  • Conexão Arduino para leitura de sensores
  • Monitoramento de cultivo de Arabidopsis thaliana

TECNOLOGIAS:

Arduino NodeJS HTML/CSS JavaScript MySQL ChartJS
